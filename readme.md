# API REST de Gestión de Libros y Autores

## 📖 Descripción
[cite_start]API RESTful construida para gestionar el flujo de información mediante operaciones CRUD completas[cite: 346]. [cite_start]Este proyecto demuestra el manejo sólido de relaciones entre tablas en bases de datos relacionales, ilustrando específicamente la relación "uno a muchos" (donde un autor tiene muchos libros asociados)[cite: 346].

## 🛠️ Tecnologías Utilizadas
* [cite_start]**Lenguaje:** C# [cite: 345]
* [cite_start]**Framework Backend:** ASP.NET Core Web API [cite: 345]
* [cite_start]**ORM (Mapeo Objeto-Relacional):** Entity Framework Core [cite: 345]
* [cite_start]**Motor de Base de Datos:** SQL Server [cite: 345]

## 🏗️ Arquitectura y Mejores Prácticas
[cite_start]La aplicación está construida demostrando un claro entendimiento de la **arquitectura basada en capas**[cite: 346]. [cite_start]A nivel estructural, implementa patrones de diseño modernos y escalables como el patrón **MVC** y el **Patrón Repositorio** para garantizar el desacoplamiento en el acceso a datos[cite: 346].
