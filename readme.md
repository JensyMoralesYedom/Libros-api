# Sistema de Ventas e Inventario

## 📖 Descripción
[cite_start]Aplicación de escritorio para la gestión integral de ventas, inventario y facturación, desarrollada como proyecto universitario[cite: 613]. [cite_start]Este sistema controla el flujo operativo completo, desde la ejecución de la venta hasta el ajuste automático del inventario y la generación de facturas digitales[cite: 616].

## ✨ Características Principales
* [cite_start]**Módulos del sistema:** Gestión de usuarios, clientes, productos y facturas[cite: 614].
* [cite_start]**Seguridad:** Login integrado con control de acceso seguro[cite: 614].
* [cite_start]**Control de stock:** Procesos de inventario sincronizados automáticamente con el registro de ventas[cite: 614, 616].

## 🛠️ Tecnologías Utilizadas
* [cite_start]**Lenguaje Base:** C# [cite: 612]
* [cite_start]**Interfaz Gráfica:** WinForms [cite: 612]
* [cite_start]**Base de Datos:** SQL Server [cite: 612]

## 🏗️ Arquitectura
[cite_start]El proyecto fue implementado utilizando Programación Orientada a Objetos (POO) y una **arquitectura en capas** que separa de forma limpia la Presentación, la Lógica de Negocio y el Acceso a Datos[cite: 615]. [cite_start]Para la persistencia, optimización y seguridad de la información, se hace uso de **procedimientos almacenados** directamente en SQL Server[cite: 615].
